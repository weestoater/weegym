# Prompt 3: Accessibility & Visual Design

Enhance the syn tracking interface with accessibility-first design:

## Requirements:

- Use color-coded progress indicators with DISTINCT patterns (not just colors) for colorblind users
- Implement semantic HTML with proper ARIA roles and labels
- Ensure full keyboard navigation support
- Add high-contrast mode support
- Test with VoiceOver (iOS) and TalkBack (Android) screen readers
- Use large, readable fonts with proper heading hierarchy
- Provide text alternatives for all visual indicators
