# Prompt 2: Meal Recommendation System

Implement a meal suggestion feature that recommends next meals based on:

## Key Factors:

- Remaining daily syn allowance
- Time of day (breakfast, lunch, dinner, snacks)
- Previously logged meals today
- Suggest "Free Food" options when syn allowance is low
- Highlight "SP (Speed Foods) days" as alternatives
- All suggestions must be accessible via screen readers with clear ARIA labels
